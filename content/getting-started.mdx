# Getting Started

Welcome to the docs! This is a simple markdown documentation site built with **Next.js** and **MDX**.

## Installation

To get started, clone the repository and install dependencies:

```bash
bun install
bun dev
```

## Writing Content

All documentation lives in the `content/` directory as `.mdx` files. Each file needs frontmatter:

```yaml
---
title: My Page Title
description: A short description.
order: 1
---
```

## Features

- **MDX Support** — Write markdown with embedded React components
- **Dark Mode** — Clean, dark-themed design
- **Table of Contents** — Auto-generated from headings
- **Sidebar Navigation** — Lists all docs pages
